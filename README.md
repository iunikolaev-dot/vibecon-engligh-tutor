# VibeCon - English Tutor

A voice-based conversational English tutor powered by OpenAI's Realtime API.

## Features

- Real-time voice conversations with AI tutor
- Specialized for upper-intermediate English learners
- Focus on business English, marketing, and lifestyle topics
- Phrasal verbs and natural expressions practice
- Built with Next.js 14 and OpenAI Realtime API

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env.local` file with your OpenAI API key:
```
OPENAI_API_KEY=your_openai_api_key_here
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

This app is ready to deploy to Vercel:

1. Push to GitHub
2. Import the repository in Vercel
3. Add your `OPENAI_API_KEY` environment variable in Vercel project settings
4. Deploy!

## Requirements

- Node.js 18+
- OpenAI API key with Realtime API access
- Modern browser with WebRTC support

## How to Use

1. Click "Start Conversation"
2. Allow microphone access when prompted
3. Start speaking in English
4. The tutor will respond with corrections, tips, and follow-up questions
